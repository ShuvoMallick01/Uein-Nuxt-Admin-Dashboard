Quick Start

Uno - Nuxt Admin Dashboard Template

Uno is a fully professional, minimal design admin dashboard template built with Nuxt 3. It features a comprehensive set of pages, components, and ready-to-use applications, leveraging the power of SSR (Server-Side Rendering) and CSR (Client-Side Rendering) for optimal performance. With its responsive layout and advanced features, Uno provides everything you need to kickstart your web application development with Nuxt 3, making it the perfect choice for modern web applications.

Key Features
  • Extensive Pages and Components: Uno offers a wide range of pre-designed pages and components, fully optimized for Nuxt 3, enabling rapid development and seamless integration into your admin dashboard.
  • Ready-to-Use Applications: Included with Uno are various fully-functional, ready-to-use applications, tailored to enhance productivity and speed up the development cycle for your admin dashboard.
  • Optimized for Nuxt 3: Each page and component is built with Nuxt 3 best practices, leveraging SSR (Server-Side Rendering) and CSR (Client-Side Rendering) for exceptional performance, ensuring your web application is both fast and scalable.
  • Responsive and Mobile-Friendly: Uno is fully responsive, providing a consistent user experience across desktop, tablet, and mobile devices, making it ideal for modern, cross-platform web applications.
  • Customizable and Scalable: With a flexible structure and modern design, Uno can be easily customized to match your unique branding, ensuring your dashboard grows alongside your application needs.
  • Integrated Developer Tools: Uno integrates seamlessly with Nuxt 3 development tools and ecosystem, offering a streamlined setup with powerful features to support your development process.
Requirements

Make sure that you have the last stable Node.js and npm version.

  • Do not delete the package-lock.json / yarn.lock file
Install

Navigate to the project root folder using the terminal and install the dependencies 

npm install

or

yarn install
Start

After the installation is complete, you can launch your app by running: 

npm run dev

or

yarn dev
Build
npm run build

or

yarn build
